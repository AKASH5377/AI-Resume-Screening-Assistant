# AI Resume Screening Assistant

## Overview

Developed an AI-powered resume screening system using LLM-based analysis.

## Features

- Evaluates candidate profiles against job requirements
- Uses prompt engineering techniques for profile assessment
- Generates automated candidate suitability reports
- Performs skill-gap analysis
- Improves screening efficiency through AI-assisted evaluation workflows

## Technologies Used

- Python
- Gemini API
- NLP
- Prompt Engineering

## Project Description

Developed an AI-powered resume screening system using LLM-based analysis. Implemented prompt engineering techniques to evaluate candidate profiles against job requirements. Generated automated candidate suitability reports and skill-gap analysis. Improved screening efficiency through AI-assisted evaluation workflows.
