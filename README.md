# AI Resume Screening Assistant

## Overview

AI Resume Screening Assistant is a Generative AI-powered application that analyzes resumes using Google's Gemini API. The system evaluates candidate profiles against job requirements, identifies skill gaps, and generates automated suitability reports to streamline the recruitment process.

---

## Features

* Evaluates candidate profiles against job requirements
* Uses prompt engineering techniques for profile assessment
* Generates automated candidate suitability reports
* Performs skill-gap analysis
* Improves screening efficiency through AI-assisted evaluation workflows
* Provides structured recommendations based on candidate qualifications

---

## Tech Stack

* Python
* Google Gemini API
* Natural Language Processing (NLP)
* Prompt Engineering

---

## Project Description

Developed an AI-powered resume screening system using Gemini API and LLM-based analysis. Implemented prompt engineering techniques to evaluate candidate profiles against job requirements. Generated automated candidate suitability reports and skill-gap analysis. Improved screening efficiency through AI-assisted evaluation workflows.

## How to Run

### Clone the Repository

```bash
git clone https://github.com/AKASH5377/AI-Resume-Screening-Assistant.git
```

### Navigate to the Project Directory

```bash
cd AI-Resume-Screening-Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Gemini API Key

```python
genai.configure(api_key="YOUR_GEMINI_API_KEY")
```

### Run the Application

```bash
python resume_screening.py
```
